Welcome!

Hello, I am Joseph Robles. I love chasing down technical projects and learning something new from them. This site was designed to showcase my radio projects in such a way that everyone could learn something new. My interests tend to center around computing, basic DC electronics, and amateur radio. As such, most of my projects have centered around these subjects.

AllstarLink Node

One of my favorite projects that I have completed so far is my Allstar node. Allstar is a Radio Over Internet Protocol program that allows radios to communicate over a network, a lot like making phone calls but between different radios. Building this node was a big project and I learned a lot!

Network Attached Storage Server

I also have made a Network Attached Storage (NAS) server for my home network. I took a used office PC, some Hard Disk Drives, Linux, and more time than it should have taken, and made a functioning storage server to hold all my movies.
